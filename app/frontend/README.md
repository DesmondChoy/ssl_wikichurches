# Frontend Notes

This directory contains the React + TypeScript + Vite frontend for the SSL WikiChurches analysis app.

- Start with the root `README.md` for setup, app routes, and the current product workflow.
- Use `app/AGENTS.md`, the repo `AGENTS.md`, and `CLAUDE.md` for implementation guidance.
- The most important current pages live under `src/pages/`: `Home`, `ImageDetail`, `Compare`, `Dashboard`, `Q2`, and `Q3`.
- Q3 follows a Dashboard-first flow: discover patterns on `Dashboard`, inspect one example on `ImageDetail`, and use `/q3` only for advanced side-by-side model comparison.
- Do not treat this file as the primary source of product behavior; keep current behavior in sync with the root docs and live code.
